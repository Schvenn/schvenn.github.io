PowerShell Development Speed Run!

I'm working on a DLP project for a large corporation right now, but this is a short-term contract and I will therefore need work after this is done. So, I'm levelling up some of my skills at the same time. I’ve started with automation and PowerShell scripting and I’ve certainly learned a lot. Here’s what I’ve accomplished in the last month:

Archives: A utility to backup and restore the current child directory structure with user configurable settings for basic scheduling, exclusions and maximum archive retention.

Assets: A utility to view scripts, commands and functions within PowerShell, in a contextualized manner, on screen, or open them for editing. Extremely useful for development purposes.

Caesar: A module to expand the basic Caesar cipher functionality to rotate alphabetic characters within a file by 1 through 25 times and add additional obfuscation techniques to plaintext files for basic encryption. Files or strings can be accepted and output. Files can be decrypted and a brute-force method can be applied to sample strings within files, using an English dictionary to find the most probable match and decrypt the files based on the results.

Compreschon: A unique encryption technique that goes far beyond the Caesar cipher, using a user customizable library for encryption and decryption, acting as an innocuous pre-shared key. Extremely useful for sharing data or code with optimal security. This is a useful cybersecurity tool for fast, secure file sharing.

DecodeSource: A cybersecurity tool to locally decode strings or files and optionally save the output. Decodes 10 different encryption techniques and features a heuristic auto-detection mode to find the most probable match. Ensures sensitive data remains private by avoiding online decoding services, which is critical for secure incident response workflows. This is a must-have for cybersecurity investigations.

FileViewer: A cybersecurity tool to view files on screen with search term functionality. Accepts single files, or can provide a selection menu, can restrict files to just plaintext documents, can accept an array of files from another source as well as search terms, which is extremely useful in automation for receiving output from another tool.

FindIn: This cybersecurity tool started as a basic text search tool, but expanded to include recursive directory searches, a file selection menu and user-configurable library of Regex search terms, which currently captures 25 different patterns of PII and other useful identifiers. This also is a must have for effective cybersecurity investigations.

GetMyCommands: A set of 4 utilities to display information about user installed functions and aliases. Useful for development.

GoodHistory: A significant improvement over PowerShell’s native command history by deduplicating it and refreshing it with only successful commands, removing artefacts such as mistyped commands. Also features an improved viewer.

HelpText: A module to allow stand-alone or plug-in help menus for PowerShell functions. Simplifies and standardizes development.

Logging: A module to improve PowerShell logging by instituting archival limits, cleaning transcripts, adding clear command and response separators and appending error logs to each session. It also includes an advanced file viewer which sorts by date and includes search functionality.

Navigation: This is the module that started it all. It comes with several functions to make navigating via the command line in PowerShell much faster. They include:

Goto, which allows fast auto-completion and directory traversal, featuring a directory cache.

feature for increased speed.

Locations, which allows user-configurable single command jumps to common locations.

Bookmarks, similar in nature to Locations, but provides a menu system of common locations for

quicker navigation than through typing entire paths.

Navigation, this module is so large, that I had to create its own help menu function.

Recents, provides a menu of the most recently navigated directories, useful for fast retracing

steps during extended PowerShell sessions.

Reload: A tool to reload modules, commands or PowerShell sessions in an intelligent manner, on the fly. A must have for PowerShell development.

Secure: A full-fledged password manager with AES256 bit encryption and a PBKDF2 (Password-Based Key Derivation Function 2) master password. Includes key and database creation and encryption, password retrieval, creation and deletion. Since it’s local and doesn’t rely on external libraries, it eliminates the need for online storage and makes data compromise far more unlikely than with any other method.

TCX2CSV: A personal project to read TCX JSON files from sources, such as Strava, extracting data, calculating new fields and exporting them to a single file, useful for import into personal run trackers. While very specific, it demonstrates the flexibility of PowerShell to extract and manipulate JSON data for use in other sources.

VTGet: A cybersecurity tool for interacting with the VirusTotal API to retrieve malware analysis data regarding URLs and files. The file mechanism calculates hashes locally, thereby maintaining the security of sensitive data and only shares the hash with VirusTotal for evaluation.

Well, it’s certainly been a busy month. I’ve kind of run out of ideas for new tools I can create in PowerShell, though. So, if nothing new comes to mind, I might take a breather and move onto improving my Python and BASH skills.
