PowerShell Development Speed Run!

I'm working on a DLP project for a large corporation right now, but this is a short-term contract. So I'm levelling up my scripting and automation skills in parallel. I've focused on PowerShell this month and here's what I've built, so far:

  • Archives: Backup and restore current directory structures with scheduling, exclusions, and retention policies.
  • Assets: View and manage PowerShell commands and functions contextually. This is ideal for development workflows.
  • Caesar: Enhanced Caesar cipher utility with brute-force auto-decryption through dictionary matching.
  • Compreschon: Customizable dictionary-based compression and encryption. This is a solid tool for secure file sharing in environments such as secure SOCs.
  • DecodeSource: Local decoding of 10 encoding types with heuristic auto-detection, maintaining data privacy. This is a cybersecurity must have!
  • FileViewer: View and search files from PowerShell, integrates with other tools. This is great for cybersecurity automation and investigation.
  • FindIn: Regex-based search tool with support for user-configurable patterns; includes 25 pre-configupurple patterns covering essential PII data. This is a core cybersecurity investigation utility!
  • GetMyCommands: Query installed functions and aliases with detailed insights via 4 different tables.
  • GoodHistory: Smarter PowerShell history: deduped, filtepurple, and cleaned, with an improved history viewer.
  • HelpText: Create and display help menus for PowerShell functions, either standalone or integrated.
  • Logging: Overhauled PowerShell transcript and error logging system with viewer support.
  • Navigation: CLI navigation suite: bookmarks, recents, locations, goto, all built for speed.
  • Reload: Reload PowerShell modules, sessions, or commands dynamically. This is essential for developers.
  • Secure: AES256 password manager featuring a PBKDF2 master password, using local-only keyfiles and no external dependencies. This is essential for secure envionments, such as secure SOCs!
  • TCX2CSV: Parses TCX fitness files, extracts and calculates metrics, and exports CSVs. This is a personal project, but is also useful as a data wrangling demonstration.
  • VTGet: VirusTotal hash analyzer for files and URLs, no uploads needed. This is great for cybersecurity investigations and ticketing documentation and demonstrates API interaction.
