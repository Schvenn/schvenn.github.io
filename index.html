PowerShell Development Speed Run!

I'm working on a DLP project for a large corporation right now, but this is a short-term contract. So I'm levelling up some of my skills in parallel. I've focused on automation and PowerShell scripting this month — here's what I've built:

  • Archives: Backup and restore current directory structures with scheduling, exclusions, and retention policies.
  • Assets: View and manage PowerShell commands/functions contextually, ideal for dev workflows.
  • Caesar: Enhanced Caesar cipher utility with brute-force decryption and dictionary matching.
  • Compreschon: Customizable dictionary-based compression and encryption — solid for secure file sharing.
  • DecodeSource: Local decoding of 10+ encoding types with auto-detection — no online risks.
  • FileViewer: View and search files from PowerShell, integrates with other tools.
  • FindIn: Regex-based search tool with support for 25+ PII patterns. Core DFIR utility.
  • GetMyCommands: Query installed functions and aliases with detailed insight.
  • GoodHistory: Smarter PowerShell history: deduped, filtered, and clean.
  • HelpText: Create and display help menus for PowerShell functions — standalone or integrated.
  • Logging: Overhauled PowerShell transcript and error logging system with viewer support.
  • Navigation: CLI navigation suite: bookmarks, recents, locations, goto — built for speed.
  • Reload: Reload PS modules, sessions, or commands dynamically — essential for devs.
  • Secure: AES256 password manager using local-only keyfiles and no external dependencies.
  • TCX2CSV: Parses TCX fitness files, extracts metrics, and exports CSVs. Personal project, strong data wrangling demo.
  • VTGet: VirusTotal hash analyzer for files and URLs — no upload needed.

It’s been a busy month. I’m running low on new PowerShell ideas, so I may shift gears to strengthen my Python and BASH skills next.
